🗂️ Bienvenido a NoteStock

Tu espacio simple y poderoso para gestionar inventario como si fuera un bloc de notas.

❓¿Qué es NoteStock?

NoteStock es una herramienta web diseñada para que puedas añadir, organizar y visualizar elementos de inventario de forma rápida y clara. Ya sea que estés administrando una colección, productos, ideas o cualquier tipo de contenido, NoteStock te ofrece un entorno limpio y eficiente.

🧩 ¿Qué puedes hacer?

  • Crear ítems fácilmente con:
    • 📛 Nombre
    • 📝 Descripción
    • 🖼️ Imagen de portada (cover)
    • 🖼️ Galería de imágenes opcional
  • Editar o eliminar ítems en cualquier momento.
  • Visualizar todos tus elementos de forma ordenada y visual.
  • Ideal para uso personal o como herramienta de apoyo para pequeños inventarios, catálogos, colecciones o ideas.
